# 커리큘럼 
스프링 입문 - 코드로 배우는 스프링 부트, 웹 MVC, DB접근 기술

- 스프링 프로젝트 생성
- 스프링 부트로 웹 서버 실행
- 회원 도메인 개발
- 웹 MVC 개발
- DB 연동 - JDBC, JPA, 스프링 데이터 JPA
- 테스트 케이스 작성

# 프로젝트 설정 
- build: gradle
- java: jdk11
- ide: intelliJ
- spring: 2.7.13
